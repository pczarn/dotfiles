## dotfiles
One repository to configure them all.

## install
Clone it anywhere. Install dotfiles with `rake`.
```bash
git clone git@github.com:pczarn/dotfiles.git
cd dotfiles
rake
```
Install more configuration.
```bash
rake sublime xfce
rake pull # keep submodules updated
```
